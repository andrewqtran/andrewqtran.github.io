GAAIN Data Visualizer
A UI & UX redesign for a big data visualizer for GAAIN, the Global Alzheimer's Association Interactive Network.
UI + UX · Illustrator

This global web application allows Alzheimer's researchers to visualize big data, provided by number of data partner. The primary objective is to offer a user-friendly process with a low learning curve. A secondary objective is to showcase the different data partners.

Besides offering a cleaner interface using basic principles of design, the new design places more emphasis on usability, such as subtle guidance for work flow order, grouping of similiar tasks and elements, and heading clarity.
