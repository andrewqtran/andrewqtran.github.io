REBL - Revenue Expense Balance Ledger



Summary

REBL is a financial dashboard web application, designed to generate dynamic reports for a wide range of users and groups. The application will be implemented within the Institute of Neuroimaging and Informatics, and licensed to other departments within the School of Medicine in a later phase.

The problem

Principal investigators, project managers, and fund managers require a unified financial system that can generate customized reports. For example, principal investigators might be interested in planning personnel allocation or reviewing account portfolio, while fund managers might need to be alerted with budget errors. Traditional spreadsheet documents can be heavy, static, and difficult to use.

Working with a project manager and a developer, I was tasked with designing the dashboard interface and data visualization.
